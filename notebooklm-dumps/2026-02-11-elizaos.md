## Architecture

### Unified Message Bus
```
Discord, Telegram, X, HTTP, onchain → Unified Event Pipeline
```
Ship agent once, swap transport adapters without touching business logic.

### Composable Swarms
```
Worlds (server/workspace) → Rooms (channel/DMs)
```
Each agent keeps its own context but can signal others:
- Delegation
- Consensus
- Load balancing

### Strategic Action Chaining
LLM-driven tool calls with:
- Branch actions
- Schedule actions
- Parallel execution
- Pause for user input
- Real operations (not just chat)

### Modular Architecture
Every capability as npm plugin:
- Model providers
- Vector stores
- Social networks
- Custom actions
- Hot-swap at runtime
- Vendor lock-in avoidance
